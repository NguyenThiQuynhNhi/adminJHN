Roles & Permissions

Control what each role can see and do across your agency. System roles cover most cases — create custom roles when you need finer control.

View mode
Scope rules. Each feature row carries an access level (None / Own / Agency). Own = only items the agent owns. Agency = every item across the agency. Specific actions (Create / Edit / Delete / Approve) only apply within the chosen scope.
